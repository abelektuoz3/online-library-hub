Admin Guidelines

Use the sidebar navigation to manage different layers of the Online Library Hub platform:

  • Catalog Resources: Add, edit, or delete books, e-books, and references.
  • Users: Access user profiles, suspend accounts, delete users, or promote other members to Administrator.
  • Contact Messages: Read and manage messages sent through the customer support form.
  • Announcements: Post updates that appear on the user dashboard.
